# DeepForensics UI 🔍

DeepForensics, sinyal işleme ve adli analiz süreçlerini görselleştirmek ve yönetmek için geliştirilmiş web tabanlı bir arayüzdür. Bu proje, `signal_project` ekosisteminin kullanıcı arayüzü (frontend) bileşenini oluşturur.

## ✨ Özellikler
* Gerçek zamanlı sinyal veri görselleştirmesi.
* Adli analiz sonuçlarının raporlanması.
* Modern ve kullanıcı dostu arayüz tasarımı.
* Hızlı geliştirme ortamı (Vite destekli).

## 🛠️ Ön Gereksinimler

Projeyi çalıştırmadan önce sisteminizde **Node.js** yüklü olduğundan emin olun.

## Kurulum
* Repoyu klonlayip projeyi calistirmak icin:
    * cd  komutu ile klonlayacaginiz  klasore girin. Ardindan cd deepforensics-ui ile  ana klasorun icine girin.
    * npm install    komutunu calistirin
    * npm run dev komutunu calistirin
    * http://localhost:5173 adresine gidin.
* Aktif bir API kullanmadigimiz icin su anda demo asamasinda  localden calistiriyoruz, ilerleyen zamanda  Railway  kullanarak projeye online olarak  erisebilecegiz.
